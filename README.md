# phpSimpleProxy
头脑简单四肢不发达的简易php反向代理程序

## 用法

https://yourdomain/proxy.php?url=someurls


## 部署

本产品只有一个文件，不需要数据库(至少现在是这样)

因此，你可以直接从release中下载这个.php文件，然后上传到你的网站进行使用

这里推荐一波我自己用BY分销做的免费主机：http://iruanpfreehosting.ml
